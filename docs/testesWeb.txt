O selenium usa um Driver para se comunicar  com o browser

código (selenium) ----------------> Driver----------->Browser

Existe um driver para cada versão ou conjunto de versões de um navegador
No caso Chrome /Edge: 1 para cada versão
Nos demais , normalmente , 1 versão de driver funciona para varias do navegador

Ha 3 formas de configurar / utilizar um driver:
- Download manual e instala dentro do projeto - Bom :Portatil - Ruim: Manutenção
- Download  manual e instala e uma pasta  compartilhada - Bom -Manutenção
- Gerenciador de download automático -  Bom -Manutenção + Portatil - Ruim: Usa recursos
